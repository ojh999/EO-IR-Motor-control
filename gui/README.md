# Pan/Tilt UDP GUI

Pan/Tilt 짐벌 시스템을 제어하는 Python tkinter 기반 GUI입니다.  
Thor(ROS2)와 UDP로 통신하며 모터 상태 모니터링 및 수동/자동/추적 제어를 지원합니다.

---

## 실행 방법

**의존성**
- Python 3.x
- tkinter (기본 내장)

**실행**
```bash
python3 gui.py --ip <Thor IP> --cmd-port 3000 --tlm-port 5001
```

**옵션**

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| `--ip` | 127.0.0.1 | Thor IP 주소 |
| `--cmd-port` | 3000 | GUI → Thor 명령 송신 포트 |
| `--tlm-port` | 5001 | Thor → GUI 피드백 수신 포트 |

## 패킷 구조

### 명령 송신 (GUI → Thor, 9B)

| 인덱스 | 필드 | 크기 | 설명 |
|--------|------|------|------|
| [0] | auto_manual | 1B | 0=scan, 1=manual |
| [1] | tracking | 1B | 0=비추적, 1=추적 |
| [2] | btn | 1B | 버튼 비트마스크 |
| [3~4] | pan | 2B | 팬 목표 위치 (Little-Endian, 0~4095) |
| [5~6] | tilt | 2B | 틸트 목표 위치 (Little-Endian, 0~4095) |
| [7] | scan_step | 1B | 스캔 스텝 (1~10) |
| [8] | manual_step | 1B | 수동 스텝 (1~10) |

**모드 조합**

| auto_manual | tracking | 최종 모드 | 동작 |
|-------------|----------|-----------|------|
| 0 (scan) | 0 (비추적) | SCAN | 팬 자동 왕복 |
| 1 (manual) | 0 (비추적) | MANUAL | 버튼으로 수동 조작 |
| 1 (manual) | 1 (추적) | TRACK | PID 추적 제어 |

### 피드백 수신 (Thor → GUI, 36B)

| 인덱스 | 필드 | 크기 |
|--------|------|------|
| [0~17] | PAN 모터 데이터 | 18B |
| [18~35] | TILT 모터 데이터 | 18B |

**모터 데이터 18B 구조**

| 오프셋 | 필드 | 크기 |
|--------|------|------|
| +0 | Moving | 1B |
| +1 | Moving Status | 1B |
| +2~3 | PWM | 2B |
| +4~5 | Current | 2B |
| +6~9 | Velocity | 4B |
| +10~13 | Position | 4B |
| +14~15 | Voltage | 2B |
| +16 | Temperature | 1B |
| +17 | Hardware Error | 1B |

---

## 버튼 비트마스크 (btn 필드)

```
bit 0 (0x01) : 팬  + (우, →)
bit 1 (0x02) : 팬  - (좌, ←)
bit 2 (0x04) : 틸트 + (업, ↑)
bit 3 (0x08) : 틸트 - (다운, ↓)
bit 4 (0x10) : 홈/센터
```

- SCAN 모드 버튼 입력 → `scan_btn`에만 반영, `manual_btn=0x00`
- MANUAL 모드 버튼 입력 → `manual_btn`에만 반영, `scan_btn=0x00`
- TRACK 모드 → 버튼 입력 없음 (`0x00`)

---

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `1` | Scan 모드 |
| `2` | Manual 모드 |
| `←` / `A` | 팬 - (좌) |
| `→` / `D` | 팬 + (우) |
| `↑` / `W` | 틸트 + (업) |
| `↓` / `S` | 틸트 - (다운) |
| `C` | 홈/센터 (pan=2048, tilt=2048) |
| `+` / `=` | 현재 모드 스텝 증가 |
| `-` | 현재 모드 스텝 감소 |

## 화면 구성

```
┌─ Mode ──────────────────────────────┐
│  [0: Scan]  [1: Manual]  □ 추적     │
├─ Direction / Home ──────────────────┤
│           ▲                         │
│       ◄   ●   ►                    │
│           ▼                         │
│  TX 9B: mode=0 btn=0x00 pan=2048...│
├─ Pan/Tilt Command Position ─────────┤
│  Pan:  [2048]  0~4095               │
│  Tilt: [2048]  0~4095               │
├─ Step Size ─────────────────────────┤
│  Scan:   [−] [ 1] [+]  1~10        │
│  Manual: [−] [ 1] [+]  1~10        │
├─ Thor -> GUI Telemetry 36B ─────────┤
│  Pan  pos / tilt pos / temp / err   │
├─ Log ───────────────────────────────┤
│  [HH:MM:SS] 이벤트 로그             │
└─────────────────────────────────────┘
```

## 주요 동작

- **명령 전송 주기**: 50ms (버튼 누르고 있을 때만 주기 전송, 아니면 이벤트 발생 시)
- **피드백 수신 주기**: 100ms UI 갱신
- **Pan/Tilt 입력**: 숫자 직접 입력 후 `Enter`로 전송
- **추적 체크박스**: ON 시 `tracking=1` 플래그 설정, 모드는 manual 유지
